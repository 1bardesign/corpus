corpus: tiny compression for strings

Corpus is a small string compression library with accompanying codebook generation script.

It is available under the MIT license on github.

Corpus stores patterns in a known set of data in a dictionary outside of the compressed data - this means that while it does a fairly bad job compared to other algorithms "in the general case", it can achieve good compression for repetitive protocol-style data and natural language, given the right dictionary for the job.
